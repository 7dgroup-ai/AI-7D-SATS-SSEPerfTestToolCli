# Change: 增强报告显示功能

## Why
当前报告存在以下问题：
1. 趋势图一行一张，占用过多空间，影响浏览效率
2. 缺少重要指标：数据块数量、平均响应时间、成功率等汇总卡片
3. 详细统计表缺少百分位数（P90/P95/P99）
4. 缺少 RPS（每秒请求数）趋势图
5. 7DGroup LOGO 设计较简单
6. 图表缺少交互功能（拖拽放大/重置）
7. 横坐标时间戳格式已正确（时分秒），但需确认一致性

## What Changes
1. **汇总卡片增强**：添加数据块数量、平均响应时间、成功率指标卡片
2. **趋势图布局优化**：修改为一行两列布局（grid），减少页面长度
3. **新增 RPS 趋势图**：显示每秒请求数的变化趋势
4. **7DGroup LOGO 重新设计**：使用 SVG 实现炫酷渐变动画效果
5. **详细统计表增强**：增加 P90、P95、P99 百分位数列
6. **图表交互功能**：使用 Chart.js zoom 插件实现拖拽放大和重置功能

## Impact
- Affected specs: report-generation（新建）
- Affected code:
  - `report_generator.py`: 主要修改 HTML 模板和 CSS/JS 部分
